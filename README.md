# Sử dụng Queue để tạo bộ đọc nội dung tài liệu HTML
## 1. Giới thiệu
Ứng dụng đọc nội dung tài liệu HTML sử dụng cấu trúc dữ liệu Queue
## 2. Tính năng chính
- Xử lý dữ liệu HTML theo đúng thứ tự xuất hiện.
- Tách và chuyển đổi nội dung HTML thành văn bản dễ đọc.
- Quản lý luồng xử lý một cách đơn giản và hiệu quả.
- Xây dựng giao diện người dùng trực quan và dễ tiếp cận.
## 3. Công nghệ sử dụng
- C#
- WinForms
- Queue
## 4. Cách cài đặt
1. Tải chương trình bằng cách truy cập nút Code, sau đó chọn Download Zip
2. Truy cập file ZIP vừa tải trong File Explorer trong Quick access dưới mục Recent files hoặc theo địa chỉ đã chọn, sau đó giải nén file
3. Sau khi giải nén, mở folder, sẽ có một file loại Application có tên “DSA”, nhấp chuột vào file để mở và bắt đầu sử dụng chương trình

## 5. Cách dùng
- Để sử dụng chức năng HTML From Test, lựa chọn một chuỗi HTML bất kỳ mong muốn được dịch, Ctrl + C chuỗi HTML rồi sau đó Ctrl + V vào hộp thoại của ứng dụng

- Để sử dụng chức năng HTML From File, hãy mở File Explorer, tìm file mong muốn được xử lý và giữ Shift + Chuột phải vào file để hiện lên “Copy as path”.
Sau khi đã thực hiện thao tác trên, nhấn chuột vào khung hộp thoại của ứng dụng và Ctrl + V để địa chỉ của file hiện ra

- Sau khi đã hoàn thành các bước trên, nhấn nút Process và ứng dụng sẽ cho kết quả
